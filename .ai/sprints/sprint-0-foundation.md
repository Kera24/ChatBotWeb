# Sprint 0: Foundation

Source: `implementation-pack/00_Operating_Model/02_Sprint_Plan.md`

## Goal

Create a clean repository, backend foundation, frontend foundation, local development structure, and quality baseline.

## In scope

- Monorepo structure
- Backend health endpoint and skeleton
- Frontend shell and placeholder routes
- Local development instructions
- Basic tests and quality commands
- Agent-readable project context

## Out of scope

- Product feature implementation
- Real authentication
- Database tenancy behavior beyond approved foundation work
- RAG runtime
- Widget runtime
- Analytics
- Billing
- Advanced agents

## Exit criteria

- Backend health endpoint works.
- Frontend app runs.
- Repository structure is understandable.
- Major folders have README guidance.
- `.ai/` operating system exists and points agents to source docs.

## Required checks

- No secrets.
- No unnecessary dependencies.
- No architecture drift.
- No UI implementation beyond approved placeholders.
