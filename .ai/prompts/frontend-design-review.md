# Frontend Design Review Prompt

Use this prompt for UI changes.

```text
Read `.ai/context/design-principles.md` and `.ai/agents/design-agent.md`.

Review this UI for:
- Expressionism alignment
- Product specificity
- Professional trust
- Accessibility
- Responsive behavior
- Information hierarchy
- Source, confidence, fallback, loading, empty, and error states
- Text overflow or overlap
- Keyboard navigation and focus states

Do not request decorative changes unless they improve comprehension, trust, or product character.

Return findings ordered by severity with file and line references.
```
