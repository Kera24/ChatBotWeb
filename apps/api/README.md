# API App

FastAPI backend for the ChatBotWeb / Yoranix AI Platform.

## Responsibilities

- Authentication integration
- Tenant and workspace resolution
- Organisation and workspace APIs
- Knowledge management APIs
- Chat runtime APIs
- Widget public APIs
- Analytics and audit APIs

## Planned stack

- Python
- FastAPI
- Pydantic
- SQLAlchemy or SQLModel
- Alembic
- PostgreSQL
- Redis

## Initial local command

```bash
uvicorn app.main:app --reload
```
